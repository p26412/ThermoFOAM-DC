# v0.2 result tables

This folder is intended for curated CSV tables generated by:

```bash
bash scripts/run_v02_analysis.sh --cases-root cases/parameterstudy --late-start 4000
```

Typical generated files:

```text
v02_global_timeseries.csv
v02_global_summary.csv
v02_global_summary_compact.csv
v02_flow_balance_table.csv
v02_heat_balance_diagnostic_table.csv
v02_main_findings.csv
v02_per_rack_summary.csv
v02_per_rack_hotspot_ranking.csv
```

Large raw OpenFOAM time directories should not be committed here.
